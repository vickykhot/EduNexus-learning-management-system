# EduNexus — Learning Management System

## Overview
EduNexus is a full-stack Learning Management System built using the MERN stack. It supports instructor and student roles with secure authentication, course management, and paid enrollments.

## Features
- Role-based access (Instructor / Student)
- Course creation and chapter-wise content
- Student dashboard & enrollment tracking
- Assignment and submission flow
- Clerk authentication & session handling
- Razorpay payment integration
- REST API based backend
- Modular controller–service–route architecture

## Tech Stack
Frontend: React.js  
Backend: Node.js, Express.js  
Database: MongoDB  
Authentication: Clerk  
Payments: Razorpay
